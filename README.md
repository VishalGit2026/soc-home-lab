#  Enterprise SOC Home Lab

##  Overview

This project demonstrates the design and implementation of an Enterprise Security Operations Center (SOC) Home Lab for security monitoring, log analysis, threat detection, and incident response.

The lab simulates a real-world enterprise environment using Windows and Linux systems integrated with SIEM technologies.

---

##  Objectives

- Build an Enterprise SOC Home Lab
- Centralize Windows Event Logs
- Monitor Security Events
- Detect Suspicious Activities
- Perform Threat Hunting
- Understand MITRE ATT&CK Framework
- Analyze Indicators of Compromise (IOC)

---

##  Lab Components

- Ubuntu Server
- Wazuh SIEM
- Elastic Stack (Elasticsearch & Kibana)
- Windows Server 2019
- Windows 11
- Sysmon
- VMware Workstation
