# Wazuh Installation

## Overview

This document describes the installation of the Wazuh SIEM platform in the Enterprise SOC Home Lab.

The Wazuh server was deployed on an Ubuntu Server virtual machine running in VMware Workstation.

---

## Lab Environment

| Component | Details |
|----------|---------|
| Platform | VMware Workstation |
| Operating System | Ubuntu Server 22.04 LTS |
| SIEM | Wazuh 4.x |
| Dashboard | Kibana / Wazuh Dashboard |

---

## Installation Steps

1. Created an Ubuntu Server virtual machine.
2. Updated the operating system packages.
3. Installed the Wazuh server.
4. Verified that the Wazuh Manager service was running.
5. Accessed the Wazuh Dashboard.
6. Connected Windows endpoints using the Wazuh Agent.
