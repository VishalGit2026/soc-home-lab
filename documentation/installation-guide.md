# Installation Guide

## Overview

This document describes the installation process used to build the Enterprise SOC Home Lab.

## Lab Environment

| Component | Version |
|----------|---------|
| VMware Workstation | Latest |
| Ubuntu Server | 22.04 LTS |
| Windows Server | 2019 |
| Windows 11 | 23H2 |
| Wazuh | 4.x |
| Sysmon | Latest |

## Installation Order

1. Install VMware Workstation
2. Create Ubuntu Server VM
3. Install Wazuh SIEM
4. Create Windows Server 2019 VM
5. Configure Active Directory
6. Create Windows 11 VM
7. Install Sysmon
8. Connect Windows Endpoint to Wazuh
9. Verify Log Collection
10. Configure Dashboards
